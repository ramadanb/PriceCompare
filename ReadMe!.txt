הפרויקט מומש באריכטקטורת 3 השכבות , כאשר ה DAL מומש באמצעות Entity Framework לפי Code First,
והVIEW מומש ב WinForms.

בפעם הראשונה שהמשתמש מריץ את התוכנה אז הDB נבנה וקבצי הXML מפוענחים וכול המידע מתווסף ל DB.

בהשוואת מחירים אני מציג עבור כול רשת את כול החנויות ממוינים בסדר עולה לפי מחיר הסל , 
והרשתות עצמן ממוינות לפי מחיר הסל של החנות הראשונה , כלומר לפי הסל הכי זול בכול רשת.


בפרויקט ממשתי את סעיפי הבונוס הבאים :
 
1. הוספת ממשק משתמש גרפי
2.שמירת סל הקניות של המשתמש בתוך קובץ שניתן לטעון אותו מאוחר יותר
3. ניהול קטלוג המוצרים בתוך DB
4. ניהול משתמשים
5. הפקת מסמך אקסל עם טבלת השוואה וגרף השוואה
